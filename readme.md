## Docker for Nette sandbox

Setting up and running Nette sandbox on docker.

## Command

```
docker-compose up
```

## Docker services

```
http://localhost:8000 PHP 7.3.3
http://localhost:8000/sandbox/www Nette sandbox
http://localhost:8080 Adminer
```
