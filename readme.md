## Docker for Nette sandbox

Setting up and running the nette sandbox server on the docker.

## Command

```
docker-compose up
```

## Server services

```
http://localhost:8000 PHP
http://localhost:8000/sandbox/www Nette sandbox
http://localhost:8080 Adminer
```
